# Sidekiq::Asap

Proof of concept. Monkey patching Sidekiq's `raw_push` for processing `asap_` queues.

## Installation

Add this line to your application's Gemfile:

    gem 'sidekiq-asap'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sidekiq-asap
